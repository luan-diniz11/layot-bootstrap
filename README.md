# Streamly

Página de exemplo para a atividade "Construindo um Layout com Bootstrap".

O projeto
- `index.html`: página única com layout feito em Bootstrap (tema: Streamly — protótipo de streaming).

Como usar
1. Abra a pasta do projeto:

```powershell
Set-Location -Path 'c:\frontend\l-bootstrap\layot-bootstrap'
```

2. Abra `index.html` no navegador (duplo-clique) ou rode um servidor simples:

```powershell
# com Python 3
python -m http.server 8000
# acesse http://localhost:8000/index.html
```
